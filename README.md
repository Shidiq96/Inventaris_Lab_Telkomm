# Lab_Telkom

Sistem Manajemen Jadwal Ruang Laboratorium berbasis web menggunakan PHP dan MySQL.

## Fitur

* Login dengan role (admin, super_admin, dosen)
* Tambah, edit, hapus jadwal
* Manajemen ruangan
* Manajemen waktu
* Manajemen dosen (diambil dari tabel users)

## Teknologi

* PHP Native
* MySQL
* HTML, CSS, JavaScript

## Role User

* **admin / super_admin** → bisa tambah, edit, hapus data
* **dosen** → hanya melihat jadwal

## Cara Menjalankan

1. Clone / download project
2. Import database ke phpMyAdmin
3. Atur koneksi database
4. Jalankan di `localhost`

---

Project ini dibuat untuk kebutuhan manajemen jadwal Lab Telkom.
